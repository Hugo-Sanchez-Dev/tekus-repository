Prueba Técnica Tekus: Implementación CRUD de Proveedores

Solución desarrollada para la prueba técnica solicitada. La aplicación se ha construido utilizando Angular 19 (Standalone Components) y Tailwind CSS 4, enfocándose en la implementación de un flujo CRUD (Crear, Consultar, Modificar, Eliminar) para la entidad Proveedores, con comunicación RESTful segura mediante JWT.
